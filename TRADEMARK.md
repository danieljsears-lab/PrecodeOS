## Official Project Identity

“Precode OS”, “Precode”, associated names or logos if any, and `https://www.precodeos.org` identify the official project created by Dan Sears / Recode.

Apache-2.0 grants copyright and patent permissions for the code and documentation. It does not grant trademark rights or permission to present a fork, derivative, product, service, or organization as official Precode OS.

## Allowed Use

Truthful, non-confusing references are allowed, including:

- “compatible with Precode OS”
- “forked from Precode OS”
- “uses Precode OS”
- “based on Precode OS”

These uses must not imply official endorsement, ownership, project continuity, or maintainer approval.

## Forks And Derivatives

Forks and derivatives should use clearly distinct names unless Dan Sears / Recode gives written permission to use the Precode OS name as the primary project or product name.

Do not use “Precode OS” or “Precode” in a way that could confuse users about which project is official, who maintains it, or where the canonical project lives.
